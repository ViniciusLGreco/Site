Suporte a Movimentos Reduzidos e Tremores

Para pessoas com mobilidade limitada, é essencial que os elementos da interface sejam fáceis de tocar, visíveis e ofereçam resposta clara ao clique ou toque.

Exemplo: Botões Grandes com Feedback

Código-Fonte Utilizado

HTML:
<button class="botao-grande" onclick="mostrarMensagem(this)">Clique Aqui</button>

JS:
function mostrarMensagem(el) {
  document.getElementById("mensagem-movimentos").innerText =
    'Você clicou no botão: "' + el.innerText + '"';
}

function ativarComTeclado(event, el) {
  if (event.key === "Enter" || event.key === " ") {
    event.preventDefault();
    el.click();
  }
}
