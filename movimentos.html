Suporte a Movimentos Reduzidos e Tremores

Elementos interativos devem ser grandes o suficiente para facilitar o clique, evitando erros para usuários com dificuldades motoras.

Exemplo: Botões Grandes

Código-Fonte Utilizado

HTML:
<button class="botao-grande">Clique Aqui</button>
<button class="botao-grande">Outra Ação</button>

CSS:
.botao-grande {
    font-size: 24px;
    padding: 15px 30px;
    border-radius: 10px;
}
