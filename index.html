Acessibilidade na Web para Idosos

Autor: Vinícius Lanaro Greco

Orientador: Prof. Me. João Ricardo dos Santos Rosa

Instituição: Universidade do Estado de Mato Grosso (UNEMAT)

Ano: 2025

Introdução

Este projeto apresenta diretrizes para tornar websites acessíveis a idosos, garantindo uma experiência digital inclusiva.

Objetivo

Desenvolver diretrizes baseadas no WCAG 2.1 e outras referências acadêmicas para otimizar a acessibilidade digital.

Navegação

Use o menu lateral para explorar os tópicos de acessibilidade.
