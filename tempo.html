Tempo de Interação e Fluxo de Uso

Os idosos podem precisar de mais tempo para completar tarefas online. Esta funcionalidade permite ajustar o tempo conforme a necessidade do usuário.

Exemplo: Temporizador Ajustável

Tempo restante: 30 segundos

Código-Fonte Utilizado

HTML:
<p id="timer">Tempo restante: <span id="tempo">30</span> segundos</p>
<button onclick="pausarTempo()">Pausar</button>
<button onclick="retomarTempo()">Retomar</button>

JS:
let tempoRestante = 30;
let intervalo = setInterval(contagem, 1000);

function contagem() {
    if (tempoRestante > 0) {
        tempoRestante--;
        document.getElementById("tempo").innerText = tempoRestante;
    }
}

function pausarTempo() {
    clearInterval(intervalo);
}

function retomarTempo() {
    intervalo = setInterval(contagem, 1000);
}
