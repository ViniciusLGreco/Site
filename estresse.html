Redução de Estresse Cognitivo

Evite sobrecarga visual e forneça um layout claro e previsível. Permita que os usuários ativem um modo de leitura simples para melhorar a experiência.

Modo de Leitura Simples

Este é um exemplo de página com menos elementos distrativos.

Código-Fonte Utilizado

HTML:
<button onclick="ativarModoSimples()">Ativar</button>
<button onclick="desativarModoSimples()">Desativar</button>
<div id="conteudo">
    <p>Este é um exemplo de página com menos elementos distrativos.</p>
</div>

JS:
function ativarModoSimples() {
    document.getElementById("conteudo").style.backgroundColor = "#FFF";
    document.getElementById("conteudo").style.color = "#000";
}

function desativarModoSimples() {
    document.getElementById("conteudo").style.backgroundColor = "#EEEEEE";
    document.getElementById("conteudo").style.color = "#212529";
}
