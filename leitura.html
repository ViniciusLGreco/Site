Facilidade de Leitura e Consumo de Conteúdo

O conteúdo deve ser legível, com espaçamento confortável e estrutura clara. Essa prática ajuda pessoas com dificuldades visuais ou cognitivas a absorver melhor a informação.

Exemplo: Ajuste de Espaçamento

Este texto pode ter o espaçamento ajustado para facilitar a leitura. Essa técnica melhora a fluidez da leitura, especialmente para usuários com dificuldades visuais ou atenção reduzida.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ab, harum aut, cumque perferendis, facere neque rerum amet iure fugiat exercitationem pariatur.

Código-Fonte Utilizado

HTML:
<div id="texto-espacamento">
  <p>Este texto pode ter o espaçamento ajustado...</p>
</div>

<button onclick="aumentarEspacamento()">Aumentar Espaçamento</button>
<button onclick="diminuirEspacamento()">Diminuir Espaçamento</button>

JavaScript:
function aumentarEspacamento() {
  document.getElementById("texto-espacamento").style.lineHeight = "2";
}

function diminuirEspacamento() {
  document.getElementById("texto-espacamento").style.lineHeight = "1.5";
}
