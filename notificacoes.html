Notificações e Alertas Acessíveis

Notificações devem ser visuais, claras e audíveis para garantir que todos os usuários percebam as mensagens importantes.

Exemplo: Notificação Visual e por Voz

Esta é uma notificação acessível.

Código-Fonte Utilizado

HTML:
<button onclick="mostrarNotificacao()">Mostrar Notificação</button>
<div id="notificacao" class="notificacao" aria-live="assertive">
  Esta é uma notificação acessível.
</div>

JS:
function mostrarNotificacao() {
  const el = document.getElementById("notificacao");
  el.style.display = "block";

  const texto = el.innerText;
  const voz = new SpeechSynthesisUtterance(texto);
  window.speechSynthesis.speak(voz);

  setTimeout(() => {
    el.style.display = "none";
  }, 4000);
}

function ativarTeclado(e, el) {
  if (e.key === "Enter" || e.key === " ") {
    e.preventDefault();
    el.click();
  }
}
