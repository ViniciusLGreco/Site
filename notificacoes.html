Notificações e Alertas Acessíveis

Notificações devem ser claras, visíveis e acessíveis a todos os usuários.

Exemplo: Notificação Visível

Esta é uma notificação acessível.

Código-Fonte Utilizado

HTML:
<button onclick="mostrarNotificacao()">Mostrar Notificação</button>
<div id="notificacao" class="notificacao">Esta é uma notificação acessível.</div>

JS:
function mostrarNotificacao() {
    let notificacao = document.getElementById("notificacao");
    notificacao.style.display = "block";
    setTimeout(() => { notificacao.style.display = "none"; }, 3000);
}
