Tipografia e Textos

A leitura acessível para idosos exige fontes adequadas, bom espaçamento e alto contraste, favorecendo a compreensão e reduzindo o esforço visual.

Exemplo: Ajuste de Tamanho da Fonte

Experimente ajustar o tamanho deste texto.

Código-Fonte Utilizado

HTML:
<p id="texto-ajustavel">Experimente ajustar o tamanho deste texto.</p>
<button onclick="aumentarFonte()" aria-label="Aumentar tamanho da fonte">Aumentar Fonte</button>
<button onclick="diminuirFonte()" aria-label="Diminuir tamanho da fonte">Diminuir Fonte</button>

JS:
function aumentarFonte() {
  document.getElementById("texto-ajustavel").style.fontSize = "22px";
}

function diminuirFonte() {
  document.getElementById("texto-ajustavel").style.fontSize = "16px";
}
