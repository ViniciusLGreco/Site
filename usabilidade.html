Testes de Usabilidade com Idosos

É essencial realizar testes de usabilidade com usuários reais para identificar barreiras, validar melhorias e tornar o site mais acessível e fácil de usar para o público idoso.

Exemplo: Feedback do Usuário

Código-Fonte Utilizado

HTML:
<textarea id="feedback" placeholder="Digite seu feedback aqui" aria-label="Área para digitar o feedback" rows="5"></textarea>
<button onclick="enviarFeedback()" aria-label="Enviar feedback">Enviar</button>
<p id="mensagem-feedback" role="alert"></p>

JS:
function enviarFeedback() {
  const mensagem = document.getElementById("mensagem-feedback");
  mensagem.innerText = "Obrigado pelo seu feedback!";
  mensagem.style.color = "green";

  // Feedback por voz para acessibilidade
  let fala = new SpeechSynthesisUtterance("Obrigado pelo seu feedback!");
  window.speechSynthesis.speak(fala);
}
