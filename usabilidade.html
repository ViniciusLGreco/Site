Testes de Usabilidade com Idosos

É essencial realizar testes com usuários reais para identificar barreiras e melhorias no site.

Exemplo: Feedback do Usuário

Código-Fonte Utilizado

HTML:
<textarea id="feedback" placeholder="Digite seu feedback aqui"></textarea>
<button onclick="enviarFeedback()">Enviar</button>
<p id="mensagem-feedback"></p>

JS:
function enviarFeedback() {
    document.getElementById("mensagem-feedback").innerText = "Obrigado pelo seu feedback!";
}
