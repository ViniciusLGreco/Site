Pagamentos e Compras Acessíveis

O processo de compra deve ser simplificado e acessível para usuários idosos, permitindo fácil preenchimento e confirmação.

Exemplo: Formulário de Pagamento

Código-Fonte Utilizado

HTML:
<label for="cartao">Número do Cartão:</label>
<input type="text" id="cartao" placeholder="0000 0000 0000 0000">
<button onclick="confirmarCompra()">Finalizar Compra</button>
<p id="mensagem-pagamento"></p>

JS:
function confirmarCompra() {
    document.getElementById("mensagem-pagamento").innerText = "Compra realizada com sucesso!";
}
