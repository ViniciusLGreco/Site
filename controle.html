Controle Total da Interface pelo Usuário

Permitir que os usuários ajustem configurações como cores, tamanhos de fonte e espaçamentos melhora a experiência.

Exemplo: Ajuste de Tema

Código-Fonte Utilizado

HTML:
<button onclick="modoEscuro()">Modo Escuro</button>
<button onclick="modoClaro()">Modo Claro</button>

JS:
function modoEscuro() {
    document.body.style.backgroundColor = "#000";
    document.body.style.color = "#FFF";
}

function modoClaro() {
    document.body.style.backgroundColor = "#F8F9FA";
    document.body.style.color = "#212529";
}
