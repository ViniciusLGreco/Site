Segurança e Privacidade

Garantir a proteção dos dados do usuário sem comprometer a acessibilidade é essencial para uma experiência segura e inclusiva.

Exemplo: Login Seguro

Código-Fonte Utilizado

HTML:
<form onsubmit="event.preventDefault(); verificarSenha();" aria-label="Formulário de Login Seguro">
  <label for="senha">Digite sua senha:</label>
  <input type="password" id="senha" placeholder="Mínimo 6 caracteres" required aria-required="true">
  <button type="submit">Entrar</button>
</form>
<p id="mensagem-seguranca" aria-live="polite"></p>

JS (em script.js):
function verificarSenha() {
  const senha = document.getElementById("senha").value;
  const mensagem = document.getElementById("mensagem-seguranca");

  if (senha.length >= 6) {
    mensagem.innerText = "Senha válida!";
    mensagem.style.color = "green";
  } else {
    mensagem.innerText = "A senha deve ter pelo menos 6 caracteres.";
    mensagem.style.color = "red";
    const aviso = new SpeechSynthesisUtterance("A senha deve ter pelo menos seis caracteres.");
    window.speechSynthesis.speak(aviso);
  }
}
