Proteção Contra Fraudes e Golpes

O site deve alertar usuários sobre golpes comuns e reforçar práticas seguras de navegação e compartilhamento de dados pessoais.

Exemplo: Alerta de Segurança

⚠️ Nunca compartilhe seus dados pessoais com terceiros. Desconfie de mensagens ou ligações suspeitas.

Código-Fonte Utilizado

HTML:
<button onclick="mostrarAlertaFraude()">Clique para ver um alerta de segurança</button>

<div id="alerta-fraude" class="alerta-fraude" role="alert" aria-live="assertive">
  <button class="fechar-alerta" onclick="fecharAlertaFraude()" aria-label="Fechar alerta">×</button>
  ⚠️ Nunca compartilhe seus dados pessoais com terceiros.
</div>

JS:
function mostrarAlertaFraude() {
  const alerta = document.getElementById("alerta-fraude");
  alerta.classList.add("mostrar");
  setTimeout(() => alerta.classList.remove("mostrar"), 8000);
}

function fecharAlertaFraude() {
  const alerta = document.getElementById("alerta-fraude");
  alerta.classList.remove("mostrar");
}
