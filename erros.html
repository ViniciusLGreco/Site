Prevenção de Erros e Confirmação de Ações

Prevenir erros ajuda a evitar frustrações. O usuário deve receber avisos antes de ações críticas, como excluir uma conta.

Exemplo: Confirmação de Exclusão

Código-Fonte Utilizado

HTML:
<button onclick="confirmarExclusao()">Excluir Conta</button>
<p id="mensagem-erro"></p>

JS:
function confirmarExclusao() {
    let confirmacao = confirm("Tem certeza que deseja excluir sua conta?");
    if (confirmacao) {
        document.getElementById("mensagem-erro").innerText = "Conta excluída com sucesso!";
    } else {
        document.getElementById("mensagem-erro").innerText = "Ação cancelada.";
    }
}
