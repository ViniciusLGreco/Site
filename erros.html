Prevenção de Erros e Confirmação de Ações

Evitar falhas e frustrações é essencial para o público idoso. Um site acessível deve alertar o usuário antes de qualquer ação crítica, como deletar uma conta ou enviar informações sensíveis.

Exemplo Prático: Confirmação de Exclusão

Código-Fonte Utilizado

HTML:
<button onclick="confirmarExclusao()">Excluir Conta</button>
<p id="mensagem-erro"></p>

JavaScript:
function confirmarExclusao() {
    const confirmacao = confirm("Tem certeza que deseja excluir sua conta?");
    const mensagem = confirmacao
        ? "Conta excluída com sucesso!"
        : "Ação cancelada.";
    document.getElementById("mensagem-erro").innerText = mensagem;
}
