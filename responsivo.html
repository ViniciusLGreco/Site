Design Responsivo e Adaptação

O site deve ser acessível em todos os dispositivos, desde celulares até desktops.

Exemplo: Teste de Responsividade

Este bloco se ajusta ao tamanho da tela.

Código-Fonte Utilizado

HTML:
<div class="caixa-responsiva">Este bloco se ajusta ao tamanho da tela.</div>

CSS:
.caixa-responsiva {
    background-color: #FFD700;
    padding: 20px;
    text-align: center;
    font-size: 18px;
    border-radius: 5px;
    width: 50%;
    margin: auto;
}

@media (max-width: 768px) {
    .caixa-responsiva {
        width: 80%;
    }
}

@media (max-width: 480px) {
    .caixa-responsiva {
        width: 100%;
    }
}
