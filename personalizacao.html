Personalização e Configurações Adaptáveis

Permitir que os usuários personalizem a experiência melhora a acessibilidade. Exemplo: ajuste de cores e tamanho da fonte.

Exemplo: Alteração de Tema e Tamanho de Fonte



Este é um exemplo de personalização de cores e tamanhos de fonte.

Código-Fonte Utilizado

HTML:
<button onclick="ativarAltoContraste()">Modo Alto Contraste</button>
<button onclick="desativarAltoContraste()">Modo Normal</button>
<button onclick="aumentarFonte()">Aumentar Fonte</button>
<button onclick="diminuirFonte()">Diminuir Fonte</button>
<p id="texto-exemplo">Texto para personalizar</p>

CSS:
#alerta-fonte {
  display: none;
  background-color: #198754;
  color: white;
  padding: 10px;
  margin-bottom: 20px;
  border-radius: 5px;
  text-align: center;
  font-weight: bold;
  animation: fadeInOut 3s forwards;
}

JS:
function ativarAltoContraste() {
  document.body.classList.add("alto-contraste");
}

function desativarAltoContraste() {
  document.body.classList.remove("alto-contraste");
}

function aumentarFonte() {
  let elemento = document.getElementById("texto-exemplo");
  let tamanhoAtual = window.getComputedStyle(elemento).fontSize;
  let novoTamanho = parseFloat(tamanhoAtual) + 2;
  elemento.style.fontSize = novoTamanho + "px";
  mostrarAlerta("Fonte aumentada!");
}

function diminuirFonte() {
  let elemento = document.getElementById("texto-exemplo");
  let tamanhoAtual = window.getComputedStyle(elemento).fontSize;
  let novoTamanho = parseFloat(tamanhoAtual) - 2;
  if (novoTamanho >= 12) {
    elemento.style.fontSize = novoTamanho + "px";
    mostrarAlerta("Fonte reduzida!");
  }
}
