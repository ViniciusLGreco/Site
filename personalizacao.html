Personalização e Configurações Adaptáveis

Permitir que os usuários personalizem a experiência melhora a acessibilidade. Exemplo: ajuste de cores e tamanho da fonte.

Exemplo: Alteração de Tema

Este é um exemplo de personalização de cores.

Código-Fonte Utilizado

HTML:
<button onclick="ativarAltoContraste()">Modo Alto Contraste</button>
<button onclick="desativarAltoContraste()">Modo Normal</button>
<p id="texto-exemplo">Este é um exemplo de personalização de cores.</p>

JS:
function ativarAltoContraste() {
    document.body.style.backgroundColor = "#000";
    document.body.style.color = "#FFF";
}

function desativarAltoContraste() {
    document.body.style.backgroundColor = "#F8F9FA";
    document.body.style.color = "#212529";
}
