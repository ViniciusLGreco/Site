Integração com Serviços Essenciais

Facilitar o acesso a serviços de saúde, pagamentos e outros essenciais torna a navegação mais segura e prática para o público idoso.

Exemplo: Acesso Rápido a Serviço de Saúde

Código-Fonte Utilizado

HTML:
<button onclick="abrirServico()" aria-label="Abrir site do serviço de saúde">Acessar Serviço de Saúde</button>
<p id="feedback-servico" aria-live="polite"></p>

JS (em script.js):
function abrirServico() {
  window.open("https://www.gov.br/saude", "_blank");
  const feedback = document.getElementById("feedback-servico");
  feedback.innerText = "Abrindo o site do serviço de saúde em uma nova guia.";
  const aviso = new SpeechSynthesisUtterance("Abrindo o site do serviço de saúde em uma nova guia.");
  window.speechSynthesis.speak(aviso);
}
