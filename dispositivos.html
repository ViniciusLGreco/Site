Compatibilidade com Dispositivos Antigos

Para garantir a acessibilidade de usuários que ainda utilizam dispositivos antigos ou navegadores desatualizados, é essencial que o site tenha código limpo, carregamento leve e verificação de recursos modernos.

Exemplo Prático: Verificação de Compatibilidade

Se você vê este texto, seu navegador é compatível.

Código-Fonte Utilizado

HTML:
<p id="compatibilidade">
    Se você vê este texto, seu navegador é compatível.
</p>

JavaScript:
if (typeof Symbol === "undefined") {
    document.getElementById("compatibilidade").innerText = "Seu navegador pode não suportar recursos modernos.";
}
