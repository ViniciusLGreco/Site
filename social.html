Engajamento Social e Inclusão Digital

O site deve promover a inclusão digital e incentivar a participação ativa dos idosos em redes sociais e outras formas de comunicação online.

Exemplo: Compartilhar Conteúdo no WhatsApp

Código-Fonte Utilizado

HTML:
<button onclick="compartilharConteudo()" aria-label="Compartilhar esta página no WhatsApp">Compartilhar no WhatsApp</button>
<p id="mensagem-compartilhamento" aria-live="polite"></p>

JS (em script.js):
function compartilharConteudo() {
  const url = encodeURIComponent(window.location.href);
  window.open("https://api.whatsapp.com/send?text=" + url, "_blank");
  
  const mensagem = document.getElementById("mensagem-compartilhamento");
  mensagem.innerText = "Abrindo o WhatsApp para compartilhamento.";

  const aviso = new SpeechSynthesisUtterance("Abrindo o WhatsApp para compartilhar esta página.");
  window.speechSynthesis.speak(aviso);
}
